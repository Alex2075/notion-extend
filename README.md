**Install:**

Firefox

1. ```about:debugging -> This Firefox -> Load Temporary Add-on...```
2. Load ```manifest.json``` from ```Firefox``` directory

Chrome

1. ```chrome://extensions -> Load unpacked```
2. Load ```Chrome``` directory
